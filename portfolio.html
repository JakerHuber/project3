Home VPN Through Azure

Last year I decided that I would build my own personal VPN. I wanted to securely connect to my home network from anywhere. First, I found an old server and switch. Then, I built a server rack out of furniture I bought from IKEA. Next, I downloaded the software I needed from Microsoft Imagine. Finally, I setup the software and configured the virtual network in Azure. Here's a diagram of my point-to-site VPN.
